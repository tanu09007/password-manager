# 🔐 Password Manager

A secure CLI-based Password Manager built with Python. Passwords are encrypted using Fernet encryption and stored locally in a JSON file.

## ✨ Features
- 🔐 Master password protection
- 🔒 Fernet encryption for all passwords
- 💾 Persistent storage using JSON
- 🎲 Random password generator
- 💪 Password strength checker
- ❌ 3 attempt limit on master password
- 🛡️ Input validation and error handling

## ⚙️ Setup
1. Clone the repo
2. Run: pip install cryptography
3. Run: python main.py

## 🛠️ Built With
- Python 3.x
- cryptography library
- hashlib
- json
